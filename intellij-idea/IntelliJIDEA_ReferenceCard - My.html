Editing
Ctrl +    Space    Basic code completion
Ctrl + Alt +    Space    Name of class completion
Alt +    Enter    Show intention actions and quick-fixes
Alt + Insert Add new file, test method, Maven depencency, ...
Ctrl + / Comment / uncomment with line comment
Ctrl + Shift + / Comment / uncomment with block comment
Ctrl + Alt + L Reformat code
Ctrl + Alt + O Optimize imports
Ctrl + Shift + ↓ / ↑ Move line under caret or selected text one block / method / class down or up
Ctrl + W Select word, line, block, ...
 midddle button + mouse move Select rectangle area of text
Ctrl + Y Delete line at caret
Ctrl + Shift + V Paste from history
Search / Replace
 2 × Shift Find everywhere
Ctrl + Shift + F Find in path
Ctrl + F Find in file
F3 Find next occurrence
Shift + F3 Find previous occurrence
Ctrl + R Replace
Alt + F7 Find usages in module
Ctrl + F7 Find usages in file
Ctrl + Shift + F7 Highlight usages in file
Refactoring
Shift + F6 Rename
Ctrl + F6 Change signature
Ctrl + Alt + M Extract method
Ctrl + Alt + V Extract variable
Ctrl + Alt + F Extract field
Ctrl + Alt + C Extract constant
Ctrl + Alt + P Extract parameter
Ctrl + Alt + T Surround with try-catch-finally, Runnable, ...
Ctrl + Alt + N Inline
Alt + Delete Safe delete
Navigation between tabs / tool windows
Ctrl + N Go to class
Ctrl + Shift + N Go to file
Alt + Home Show navigation bar
Ecs Go to editor from tool window (but don't hide tool window)
Shift + Ecs Go to editor from tool window (and hide active or last tool window)
Ctrl + Shift + F12 Maximize / minimize editor
Ctrl + PageDown / PageUp Go to next / previous editor tab
Ctrl + F4 Close active editor tab
Scroll Lock Close all non-active tabs
Pause (Break) Close all tabs
Alt + one of 0..9 Open corresponding tool window
Ctrl + Tab Switch between tabs and tool windows
Ctrl + E Switch between recent files
F12 Jump to last used tool window
Shift + F4 Open source in new editor window
 midddle button click Close tab or tool window under mouse pointer
Navigation inside editor / tool window
Ctrl + G Go to line
Alt + ↓ / ↑ Go to next / previous method
Alt + ← / → Navigate back / forward across code
F2  or  Shift + F2 Next / previous highlighted error
F6  or  Ctrl + click Go to declaration
F5  or  Ctrl + Alt + click Go to implementation(s)
Ctrl + S Equivalent to "Scroll From Source" button in Project tool window (additional plugin is involved)
Quick info
Ctrl + P Parameter info (within method call arguments)
Ctrl + Shift + I Quick definition lookup
Ctrl + Q Quick documentation lookup
Ctrl + mouse over code Brief info about code
Ctrl + F12 File structure popup

Ctrl + Shift + ↑ / ↓ Drag border (change height) of active tool window
Ctrl + B Toggle bookmark
F11 Enter / leave full screen
