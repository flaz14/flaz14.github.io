Скрипт будет простым. Т.е. все параметры будут задаваться в самом скрипте, без всяких аргументов командной строки, переменных окружения т.д. Скачивать будем все файлы (т.е. для всех архитектур, исходный код тоже). Но в первую очередь будем скачивать файлы, содержащие контрольные суммы. Почему? Потому что, каким бы не был классным wget, возможны обрывы соединения. Т.е. придется докачивать большие файлы. А чтобы быть уверенным, что все докачалось нормально, нужно перво-наперво проверить контрольные суммы. Есть еще один нюанс: во время файлы на сервере архивов Ubuntu довольно часто меняются.
